# string-utils

Small string utility functions used across the project.

## Functions (`src/strings.ts`)

- `capitalize(s)` — capitalizes the first letter of `s` and lowercases the rest.
- `truncate(s, maxLen)` — truncates `s` to at most `maxLen` characters, adding
  `"..."` if truncated.
- `reverseWords(s)` — reverses the order of words in a sentence.

See the JSDoc comment above each function for the exact contract and examples.
