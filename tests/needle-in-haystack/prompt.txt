This workspace contains a small text-processing "plugin" system under
`plugins/`. Each file in `plugins/` exports a default object implementing
the `Plugin` interface defined in `types.ts`:

  export interface Plugin {
    name: string;
    description: string;
    run(input: string): string;
  }

One of these plugins is supposed to capitalize the first letter of *every
word* in its input, lowercasing the rest of each word — e.g. turning
"hELLO wORLD" into "Hello World". Its current `run` implementation has a
bug: it only capitalizes the first letter of the *entire input* and
lowercases everything else, so "hello world" incorrectly becomes
"Hello world" instead of "Hello World", and "two three four" becomes
"Two three four" instead of "Two Three Four".

Find the one plugin with this bug and fix its `run` function so it
correctly capitalizes every word as described above. Leave its `name` and
`description` fields exactly as they are — only change the body of `run`.

Constraints:
- Modify exactly one file: the plugin with the bug described above.
- Do not modify `types.ts` or any other file under `plugins/`.
